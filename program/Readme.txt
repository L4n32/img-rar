Using:
1. Select an archive with data by clicking "Open rar".

2. Select an image by clicking "Open img", in which the data archive will be placed.

3. Select the final file name by clicking "Save file", whose extension will match the extension of the photo selected in step 2.


Использование:
1. Выберите архив с данными, нажав "open rar".

2. Выберите изображение, нажав "open img", в которое будет помещён архив с данными.

3. Выберите итоговое имя файла, у которого расширение будет совпадать с расширением выбранного во 2 пункте фото.
